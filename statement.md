# Problem Statement

Traditional banking operations face several challenges:
- Manual account management leads to human errors
- Inefficient transaction processing and tracking
- Poor customer experience with long waiting times
- Lack of real-time account information access
- Difficulty in maintaining transaction history

# Problem Statement

Students and campus staff need a simple, reliable system to manage bank accounts and transactions. This project provides a console-based Student Bank Management System for VIT Bhopal that handles account creation, deposits, withdrawals, transfers, and transaction history.

# Scope of the Project

- Implement core account and transaction features (create account, deposit, withdraw, transfer).
- Track transaction history with timestamps and balance-after values.
- Provide summary views (all accounts, all students, bank totals).
- Keep the implementation lightweight and console-based (no database or web UI in this version).

# Target Users

- VIT Bhopal students who need to manage personal accounts.
- Campus staff who help maintain student accounts.
- Instructors and students using the project for learning and demonstration.

# High-level Features

- Student account creation and management.
- Multiple account types: Savings, Current, Salary.
- Deposit, withdrawal, and transfer operations with validation.
- Transaction history with timestamps and transaction IDs.
- Bank summary: total students, total accounts, and total deposits.
- Simple console-based menu for interactive use and testing.
